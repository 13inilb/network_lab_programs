## for this things are a little different

### compile both
```bash
  cc client.c -o client
  cc server.c -o server
```
### for running server
```bash
  ./server
```
### for running client
```bash
  ./client <ip address> <File name>
```
### For the confused people here is an example :

```bash
  ./client 127.0.0.1 test.txt
```
